PROTOCOL ORDER (canonical):
  1. saddle_analysis()
     → per-gene tumour vs normal
     → predictions vs actuals
  2. build_depth_score()
     → uses confirmed genes
     → fallback to full panel if <2
  3. depth_correlations()
     → Pearson r, every panel gene
     → READ THIS FIRST — THE DISCOVERY
  4. gap_tests()
     → circuit topology
     → BROKEN = intervention point
  5. subtype_analysis()
     → Type 1 vs Type 2 vs CIMP depth
     → tests S1-P1
  6. quartile_analysis()
     → Q4/Q1 gene ratios
  7. drug_targets()
     → stated before literature

DISCOVERY PRINCIPLE:
  Panel is wide (70+ genes).
  Unexpected signals are the finding.
  Confirmations validate the framework.
  Surprises extend it.
