## PREAMBLE — WHY THIS PROTOCOL EXISTS

```
The Convergence Node Score (CNS) equation
identifies positions in a pathogen proteome
that are:
  — Maximally constrained by function
    (the pathogen cannot leave them).
  — Maximally conserved across strains
    (the pathogen never has).
  — Maximally accessible to immune targeting
    (the position can be reached).

The CNS equation does not ask:
  Does the position that is maximally
  constrained in the pathogen also exist —
  by sequence, by conformation, or by
  structural analogy — anywhere in the
  host's own biology?

This question must be asked separately.
It requires its own protocol.
This is that protocol.

THE GEOMETRIC REASON THIS QUESTION IS
MANDATORY:

The framework's own logic, applied without
restriction, produces the following
dangerous inference:

  High CNS score →
  The pathogen cannot leave this position.
  Therefore: attack everything at this
  position.

But "everything at this position" includes
the host if the host occupies the same
geometric position in sequence space.

The safety protocol applies the framework's
own invariance logic to the HOST:
  If the host shares the convergence node,
  the host is also constrained to occupy it.
  An immune response against a shared
  convergence node is an immune response
  against something neither the pathogen
  NOR THE HOST can escape.
  That is the definition of an autoimmune
  trap: a position where destroying the
  target destroys the self.

The protocol prevents this outcome by
verifying exclusive pathogen ownership
of the convergence node before deployment.
```

## PART I — THE EXCLUSIVE OWNERSHIP
## PRINCIPLE: THE THEORETICAL FOUNDATION

```
DERIVATION:

A convergence node is defined by HIGH
FUNCTIONAL ESSENTIALITY.
The pathogen cannot leave this position
because leaving it destroys function.

Apply the same logic to the host:
If the HOST shares this position,
the host is also constrained to occupy it.
Functionally essential positions are
conserved not just in pathogens but in
any organism for which the function is
also essential.

CASE A — THE POSITION IS ESSENTIAL
IN BOTH PATHOGEN AND HOST:
  Both are constrained.
  Neither can escape.
  An immune response targeting this
  position attacks both.
  THIS IS THE AUTOIMMUNE TRAP.
  This is precisely what the protocol
  must detect and flag.
  A CNS-high position that is shared
  between pathogen and host is NOT
  a vaccine target.
  It is a contraindication.

CASE B — THE POSITION IS ESSENTIAL
IN THE PATHOGEN BUT NOT THE HOST:
  The pathogen is constrained.
  The host has no constraint at
  this position and has therefore
  drifted away from it over evolutionary
  time.
  The host and pathogen occupy different
  positions in sequence space.
  An immune response targeting the
  pathogen's position does not find
  the host's position.
  THIS IS THE CLEAN CONVERGENCE NODE.
  This is the vaccine target.

CASE C — THE HOST HAS ACTIVELY DRIFTED
AWAY FROM THE PATHOGEN'S CONVERGENCE NODE
OVER EVOLUTIONARY TIME:
  This is the most common and most
  important case.
  The mechanism: host-pathogen co-evolution
  generates immune responses in the host
  against whatever the pathogen presents.
  Any host cell that RESEMBLES the pathogen's
  constrained position is attacked by the
  host's own immune system in the same way
  as the pathogen.
  Over millions of generations, host cells
  that resemble the pathogen's constrained
  positions are eliminated.
  Host cells that DO NOT resemble those
  positions survive.
  Natural selection drives the host proteome
  AWAY from pathogen convergence nodes.

  THIS IS THE EXCLUSIVE OWNERSHIP PRINCIPLE:

  Pathogen convergence nodes tend to be
  exclusively pathogen-owned because host
  evolution selects the host away from any
  sequence position that the host's own
  immune system has learned to target.
  The pathogen and host co-evolve in opposite
  directions relative to the convergence node:
    — Pathogen: selected TO STAY at the node
      (functional constraint).
    — Host: selected TO LEAVE the node
      (immune pressure against resembling
      the pathogen).
  Over sufficient evolutionary time, these
  opposing pressures produce exclusive ownership.

  THIS IS NOT ABSOLUTE:
  It is a strong tendency, not a guarantee.
  The protocol exists because the tendency
  must be verified, not assumed.
  Young co-evolutionary relationships,
  recently emerged pathogens, and certain
  deeply conserved biochemical functions
  may produce exceptions.
  The protocol's function is to identify
  whether a given convergence node is
  following the expected pattern (exclusive
  ownership) or is an exception (shared
  ownership / Case A above).

THE NATURAL HISTORY INFERENCE RULE:

For pathogens with long evolutionary
co-existence with the host species
(centuries to millennia), the following
inference rule applies:

  IF: the pathogen has circulated in the
  host population long enough for natural
  infection to generate immune responses
  against the convergence node,

  AND: no population-scale autoimmune
  syndrome attributable to antibodies
  against the convergence node has been
  documented,

  THEN: the convergence node is empirically
  consistent with exclusive pathogen
  ownership at a confidence level proportional
  to the size and duration of the
  natural exposure population.

  STATEMENT: A convergence node that has
  been naturally targeted by the immune
  systems of millions of individuals per
  year, for years to decades, without
  producing a clinical autoimmune signal,
  has passed the most powerful real-world
  safety test available.

  This rule does not replace formal
  molecular analysis.
  It provides the highest-confidence
  empirical prior for the formal analysis
  to confirm.
  When the natural history inference
  and the formal molecular analysis agree:
  the safety case is strong.
  When they disagree: investigate the
  disagreement before proceeding.
```

## PART II — THE THREE-FORM MIMICRY
## ANALYSIS: THE OPERATIONAL PROTOCOL

```
Every convergence node must be evaluated
against all three forms of molecular mimicry.
Each form requires a different analysis method.
Each form has a different failure mode if missed.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

FORM 1 — LINEAR SEQUENCE MIMICRY

DEFINITION:
  The primary amino acid sequence of the
  convergence node, or any sub-segment of
  it above the threshold length for
  antibody epitope binding (typically 6-8
  continuous residues for linear B-cell
  epitopes), is present in:
    (a) The human proteome.
    (b) The proteome of any organism the
        host carries as a commensal or is
        persistently infected by.

WHY THIS FORM MATTERS:
  Antibodies that recognise linear epitopes
  will bind any protein that presents the
  same linear sequence.
  If the same 6-8 residues appear in a
  human surface protein, anti-convergence-node
  antibodies will bind human surface proteins.
  Autoimmunity follows.

ANALYSIS METHOD:
  Step 1: Extract the full convergence node
  sequence as identified by the CNS equation.

  Step 2: Perform BLAST against:
    — UniProt Human Proteome (reviewed).
    — UniProt Mouse Proteome (as animal
      model safety proxy).
    — NCBI NR database, filtered for:
        All human commensal bacteria
        (Bacteroides, Faecalibacterium,
        Lactobacillus, Bifidobacterium,
        Akkermansia, Streptococcus, and
        all genera in the Human Microbiome
        Project reference set).
        Human skin microbiome bacteria
        (Cutibacterium acnes, Staphylococcus
        epidermidis, Malassezia furfur, etc.)
        Human respiratory microbiome bacteria
        (Haemophilus influenzae, Moraxella
        catarrhalis, Streptococcus pneumoniae,
        and resident commensals).

  Step 3: For any hit with E-value < 0.01
  over the convergence node sequence:
    — Flag as FORM 1 POSITIVE.
    — Record the organism, the protein
      name, and the location on that protein.
    — Assess physiological accessibility
      (is this protein surface-exposed to
      circulating antibodies under normal
      health conditions?).
    — If accessible: CONTRAINDICATION until
      resolved by epitope engineering to
      remove the overlapping residues from
      the vaccine antigen while preserving
      immunogenicity at non-overlapping positions.
    — If inaccessible (intracellular, or only
      exposed during cell death): note as
      cryptic mimic. Flag for monitoring in
      clinical trials. Not an absolute contraindication
      but requires informed surveillance.

THRESHOLD FOR CLEARANCE:
  No hit with E-value < 0.01 over the
  convergence node sequence against any
  accessible surface protein in the above
  databases.
  Documentation: BLAST output logs preserved
  in the vaccine candidate safety file.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

FORM 2 — CONFORMATIONAL / STRUCTURAL
MIMICRY

DEFINITION:
  The three-dimensional structure of the
  convergence node, in its FOLDED OR
  MEMBRANE-ASSOCIATED CONFORMATION, is
  structurally similar to a surface-accessible
  region of a human or commensal protein
  — even in the ABSENCE of sequence homology.

WHY THIS FORM MATTERS:
  Antibodies are raised against three-dimensional
  conformational epitopes, not linear sequences.
  A protein with no sequence homology to the
  convergence node may adopt a similar surface
  conformation and be recognised by
  anti-convergence-node antibodies.
  This is conformational cross-reactivity —
  the form of mimicry least often checked and
  most often responsible for unexpected
  autoimmune reactions.

WHY THIS FORM IS HARDER TO RULE OUT:
  The complete human conformational proteome
  is too large to exhaustively screen.
  All human proteins sample a range of
  conformational states (conformational ensembles).
  A protein that transiently adopts a
  conformation similar to the convergence node
  may be targeted during that transient state,
  even if its ground-state conformation is
  entirely different.

ANALYSIS METHOD:
  Step 1: Obtain or predict the 3D structure
  of the convergence node in its physiologically
  relevant conformation.
    — Use X-ray crystallography or cryo-EM
      data if available.
    — If unavailable: use AlphaFold3 prediction
      of the full protein in the membrane context,
      extract the convergence node surface.

  Step 2: Structural similarity search against:
    — PDB (Protein Data Bank): DALI or PDBeFold
      search for structural homologs of the
      convergence node surface patch.
    — AlphaFold Human Proteome database:
      Foldseek search for proteins with similar
      surface topology to the convergence node.
    — Focus search on membrane-proximal and
      extracellular domains only (the subspace
      accessible to circulating antibodies).

  Step 3: For any structural hit with
  TM-score > 0.5 over the convergence
  node surface patch:
    — Flag as FORM 2 CANDIDATE.
    — Assess expression context: is this
      protein surface-expressed on healthy
      circulating cells, secreted proteins,
      or luminal epithelial surfaces?
    — If yes: in vitro binding assay required.
      Test anti-convergence-node polyclonal
      antibody (raised in rabbit or mouse)
      against the identified human protein
      using ELISA and/or SPR.
      If binding confirmed: FORM 2 POSITIVE.
    — If no accessible expression: note as
      cryptic conformational mimic.
      Flag for clinical monitoring.

  Step 4 (Ensemble mimicry evaluation):
    For convergence nodes that are intrinsically
    disordered or membrane-proximal:
    The conformational ensemble must be sampled,
    not just the lowest-energy structure.
    Method: Molecular dynamics simulation of
    the convergence node (minimum 100 ns in
    explicit solvent with membrane model if
    membrane-associated) to sample the
    conformational ensemble.
    Extract the 5-10 most populated distinct
    conformations.
    Screen each conformation independently
    against the human accessible proteome
    using Foldseek or equivalent.
    Any hit from any ensemble member is a
    FORM 2 CANDIDATE requiring in vitro validation.

THRESHOLD FOR CLEARANCE:
  No confirmed in vitro binding of
  anti-convergence-node antibodies to any
  human surface-accessible protein.
  Structural candidates with no binding
  activity: documented as Form 2 assessed,
  no activity detected.
  Documentation: All structural search
  outputs, binding assay results, and
  ensemble simulation data preserved
  in the vaccine candidate safety file.

EMPIRICAL FAST-TRACK CLEARANCE:
  If the pathogen has circulated in the
  host population for > 10 years AND
  natural infection generates measurable
  anti-convergence-node antibody titres
  in a significant fraction of the population
  AND no clinical autoimmune syndrome
  attributable to those antibodies has
  been described in the literature:
  Form 2 empirical clearance may be claimed,
  with the following documentation:
    — Literature review confirming natural
      anti-convergence-node antibody generation.
    — Literature review confirming absence
      of associated autoimmune syndrome.
    — Formal structural analysis still required
      and documented, even if outcome is used
      as confirmation rather than primary evidence.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

FORM 3 — ECOLOGICAL MIMICRY
(COMMENSAL AND VIROME CROSS-REACTIVITY)

DEFINITION:
  The convergence node sequence or conformation
  is shared with a surface-exposed protein of:
    (a) A commensal organism the host regularly
        harbours (gut microbiome, skin microbiome,
        respiratory microbiome).
    (b) A persistent virus the host carries
        in a non-pathological latent or chronic
        state (endogenous retroviruses, herpesviruses,
        anelloviruses, commensal bacteriophages).
    (c) Environmental organisms that are regularly
        encountered without causing disease
        (food microorganisms, soil bacteria, etc.)

WHY THIS FORM MATTERS:
  Autoimmunity is the obvious risk the
  protocol guards against.
  Ecological mimicry is a different class of risk:
    — Anti-convergence-node antibodies that
      target a commensal gut bacterium could
      selectively deplete that bacterium from
      the microbiome.
    — Selective depletion of a commensal species
      creates a niche for less beneficial species.
    — The resulting dysbiosis may produce
      downstream health effects that are not
      classically "autoimmune" but are nonetheless
      a consequence of the vaccine.
  This form is the least often evaluated and
  the most often the source of unexpected
  ecological consequences.

WHY THIS FORM IS DIFFERENT FROM FORMS 1 AND 2:
  Forms 1 and 2 protect against the immune
  system attacking the self (or a structure
  so similar to self that the distinction
  is functionally irrelevant).
  Form 3 protects against the immune system
  attacking the ecological partners of the self
  — the organisms the host depends on for
  health, which are not self but which are
  essential to the host's normal function.

ANALYSIS METHOD:
  Step 1: Form 1 analysis (BLAST) against
  commensal bacterial proteomes is already
  included in the Form 1 protocol above.
  This catches linear sequence overlaps
  with commensal bacteria.

  Step 2 (VIROME-SPECIFIC):
  BLAST the convergence node sequence against
  the human commensal virome proteome:
    — Endogenous human retroviruses (HERV
      database — ERVmap or equivalent).
    — Human persistent herpesviruses:
      HSV-1, HSV-2, EBV, CMV, VZV, HHV-6,
      HHV-7, HHV-8. Focus on proteins
      expressed during reactivation and
      expressed on the virion surface during
      productive infection.
    — Human anelloviruses (TTV, TTMV, TTMDV):
      prevalent persistent virome members.
    — Bacteriophages present in the gut virome
      (particularly those infecting major
      commensal bacteria — M13, T4, phi29
      families, and gut-specific phage families).

  Step 3 (STRUCTURAL — VIROME):
  For any BLAST hit in Step 2 with E-value < 0.01:
  Structural overlay using AlphaFold predictions.
  In vitro binding assay of anti-convergence-
  node antibodies against the virome protein.

  Step 4 (ECOLOGICAL CONSEQUENCE ASSESSMENT):
  Even if no binding is detected:
  Document the analysis.
  If binding IS detected at a commensal
  organism protein:
    — Assess whether that organism is
      essential or beneficial to the host.
    — Assess the probable ecological impact
      of antibody-mediated targeting of
      that organism.
    — If the organism is important (e.g.,
      a keystone commensal gut bacterium):
      Flag as FORM 3 POSITIVE.
      Redesign of the vaccine antigen to
      exclude overlapping residues required
      before proceeding.
    — If the organism is a persistent virus
      whose lytic reactivation is already
      controlled by immune mechanisms:
      Cross-reactivity may be neutral or
      beneficial (additional immune pressure
      against a potentially pathogenic
      reactivation). Document reasoning.

  Step 5 (IN VIVO MICROBIOME MONITORING):
  In NHP (non-human primate) safety study:
    — Baseline characterise gut, skin, and
      respiratory microbiome by 16S/ITS/virome
      metagenomics.
    — Administer the vaccine.
    — Re-characterise microbiome at 4, 8,
      and 16 weeks post-vaccination.
    — If any significant compositional shift
      in the microbiome is observed and
      correlated with anti-convergence-node
      titre: flag as FORM 3 IN VIVO SIGNAL.
    — Investigate the species affected.
    — Determine whether effect is transient
      or persistent.
    — Apply clinical safety consequence
      judgement based on the ecological
      importance of the affected species.

THRESHOLD FOR CLEARANCE:
  No confirmed binding of anti-convergence-node
  antibodies to proteins of commensals or
  persistent viruses identified as ecologically
  important.
  No significant microbiome compositional shift
  in NHP safety study.
  Documentation: All BLAST outputs, virome
  structural overlays, binding assay results,
  and NHP microbiome data preserved in
  the vaccine candidate safety file.

EMPIRICAL FAST-TRACK CLEARANCE:
  Same rule as Form 2:
  If natural infection generates anti-convergence-
  node antibodies in a large population over
  years to decades, without clinically documented
  microbiome disruption or ecological
  consequence attributable to those antibodies:
  Form 3 empirical clearance may be claimed.
  Formal virome analysis still required
  as confirmatory documentation.
```

## PART III — THE RESIDUAL EFFECTS
## ACCOUNTING FRAMEWORK

```
Beyond the three forms of mimicry, every
convergence node vaccine must account for
residual effects that do not arise from
cross-reactivity at all but that are
nonetheless consequences of the intervention
at ecological, evolutionary, or
immunological scales.

These residual effects are not safety
contraindications in the classical sense.
They are SECOND-ORDER CONSEQUENCES that
require planning, monitoring, and in some
cases mitigation by formulation design.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

RESIDUAL EFFECT CLASS 1 —
NICHE DISPLACEMENT (PATHOGEN ECOLOGY)

DEFINITION:
  When immune pressure against a pathogen's
  convergence node is globally deployed,
  the target pathogen population is reduced.
  The ecological niche previously occupied
  by the target pathogen may be filled by
  related pathogens not targeted by the
  convergence node.

  EXAMPLE:
  M2e-based influenza A vaccine reduces
  influenza A burden.
  Influenza B occupies the vacated
  respiratory virus niche.
  Influenza B is not covered by
  anti-M2e antibodies.
  Net benefit is reduced relative to
  predictions based on influenza A
  burden alone.

ASSESSMENT REQUIREMENT:
  For every convergence node vaccine:
    — Identify the closest phylogenetic
      relatives of the target pathogen.
    — Determine whether those relatives
      share the convergence node.
    — If they do: the vaccine's coverage
      is broader than the primary pathogen
      (POSITIVE residual).
    — If they do not: those relatives
      may expand ecologically as the
      primary pathogen is controlled
      (REQUIRES MONITORING).
    — Mitigation option: Design a
      multi-target formulation that
      includes the homologous convergence
      node from the closest non-covered
      relative.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

RESIDUAL EFFECT CLASS 2 —
EVOLUTIONARY ESCAPE UNDER GLOBAL PRESSURE

DEFINITION:
  Even a highly constrained convergence node
  can be escaped if the selection pressure
  is strong enough and the population size
  is large enough.
  The CNS equation scores the FITNESS COST
  of escape under current immune pressure levels.
  Global deployment of the vaccine changes
  the immune pressure level.
  A partially-escape-competent variant that
  is outcompeted at current pressure levels
  may be selected FOR at global pressure levels
  if it pays an acceptable fitness cost in the
  new competitive landscape.

EXHAUSTIVENESS PRINCIPLE APPLIED:
  "Whatever can happen will happen."
  Given a sufficiently large pathogen
  population, sufficient generations, and
  sufficient selection pressure:
  Any escape variant with non-zero viability
  will eventually emerge.
  The question is not WHETHER escape variants
  emerge but WHEN and HOW VIABLE they are.

ASSESSMENT REQUIREMENT:
  For every convergence node vaccine:
    — Identify all known natural variants at
      the convergence node across all sequenced
      strains in public databases (GISAID, NCBI).
    — For any natural variant: assess whether
      it is associated with reduced function
      or reduced fitness (literature review and
      structural modelling).
    — If natural variants exist that are viable:
      the convergence node is not fully escape-
      resistant. Increase monitoring frequency.
      Consider multi-node formulation.
    — If no viable natural variants exist:
      document the evidence and the structural
      reasoning (functional constraint mechanism).

  MITIGATION:
  Multi-target formulation with orthogonal
  convergence nodes.
  If the pathogen must escape two independently
  constrained positions simultaneously to
  avoid the immune response, the probability
  of successful escape is orders of magnitude
  lower than for single-target escape.
  RECOMMENDATION: Every convergence node
  vaccine should include at least one
  secondary convergence node from an
  orthogonal functional system in the
  pathogen (different protein, different
  functional class from the primary node).

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

RESIDUAL EFFECT CLASS 3 —
ORIGINAL ANTIGENIC SIN
(IMMUNE IMPRINTING IN THE HOST)

DEFINITION:
  In hosts with prior exposure to the
  target pathogen, memory B cells and
  antibodies exist against previously
  encountered antigens.
  When the convergence node vaccine is
  administered, the immune system may
  preferentially re-stimulate pre-existing
  memory rather than forming new responses
  to the convergence node.
  This is Original Antigenic Sin
  (also called immune imprinting):
  the immune system's tendency to respond
  to new antigens through the lens of
  prior exposure.
  If the convergence node is poorly
  immunogenic in the context of natural
  infection (as is the case for many
  convergence nodes — their functional
  constraint is precisely why pathogens
  have evolved to minimise their
  immunological visibility), then:
    — Natural infection generates strong
      memory against non-convergence-node
      antigens.
    — The convergence node vaccine must
      overcome that immunodominance.
    — Original antigenic sin may redirect
      germinal centre resources toward
      non-convergence-node memory, reducing
      anti-convergence-node titres.

ASSESSMENT REQUIREMENT:
  For every convergence node vaccine in
  a pathogen that circulates naturally:
    — Assess the immunodominance landscape:
      what antigens does natural infection
      typically generate dominant memory
      against?
    — Is the convergence node subdominant
      in the natural infection response?
      (If yes: imprinting risk is higher.)
    — Test anti-convergence-node titre
      achievement in seropositive individuals
      (prior natural exposure) vs seronegative
      individuals.
    — If seropositive individuals achieve
      significantly lower anti-convergence-node
      titres: formulation modifications required.

  MITIGATION OPTIONS:
    — High-density antigen display (nanoparticle,
      tandem repeat, VLP) to maximise
      B-cell receptor crosslinking specifically
      at the convergence node epitope,
      outcompeting lower-density memory recall.
    — Adjuvant selection: germinal-centre
      promoting adjuvants (AS01B, CpG-1018,
      ISCOM matrices) increase GC capacity
      and reduce zero-sum competition between
      new and memory responses.
    — Timing with respect to recent natural
      exposure: if possible, administer
      the convergence node vaccine during
      an interval when natural infection
      memory has waned rather than
      immediately after an acute infection.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

RESIDUAL EFFECT CLASS 4 —
IMMUNOLOGICAL CORRIDOR EFFECTS
(BYSTANDER ACTIVATION AND LINKED
EPITOPE SUPPRESSION)

DEFINITION:
  Any vaccine antigen generates responses
  not only to the intended epitope but to
  all B-cell and T-cell epitopes present
  in the vaccine formulation.
  When the convergence node is presented
  as part of a larger protein scaffold
  (ferritin, VLP, albumin-binding carrier,
  etc.), immune responses to the scaffold
  are also generated.
  These scaffold-specific responses are
  typically benign.
  However, if the scaffold shares epitopes
  with host proteins (e.g., ferritin used
  as nanoparticle scaffold for M2e display
  — ferritin is an endogenous human protein),
  bystander immune responses against the
  scaffold may have secondary effects.

  For ferritin specifically:
  Human ferritin is an intracellular
  iron-storage protein, not a surface-
  expressed protein.
  Anti-ferritin antibodies are not
  expected to cause autoimmunity because
  ferritin is inaccessible to circulating
  antibodies in healthy tissue.
  However, in conditions associated with
  elevated serum ferritin (severe infection,
  inflammatory states, haemophagocytic
  lymphohistiocytosis), ferritin IS
  detectable in the serum.
  Anti-ferritin antibodies generated by
  the scaffold response could theoretically
  form immune complexes with serum ferritin
  in these specific conditions.

ASSESSMENT REQUIREMENT:
  For every convergence node vaccine
  using a protein scaffold:
    — Identify whether the scaffold protein
      is present in the human body
      (endogenous protein vs truly foreign
      scaffold like SpyCatcher).
    — If endogenous: assess accessibility
      under normal and pathological conditions.
    — Test for scaffold-specific antibody
      generation in animal models.
    — Assess for any adverse events in
      animal models that correlate with
      scaffold-specific titre.
    — If any signal: either use a
      non-endogenous scaffold or remove
      scaffold-overlapping epitopes from
      the formulation by structural modification.
```

## PART IV — THE SAFETY GATE DECISION
## FRAMEWORK: GO / NO-GO / REDESIGN

```
After completion of all three mimicry forms
and all four residual effect classes, the
```
