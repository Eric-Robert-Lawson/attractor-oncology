```

  Arrested culture.
  LECA conditions maintained.
  No UV exposure.

  Baseline control.

Condition 3 — Modern yeast + UV:
  Germinating yeast culture.
  Standard YPD medium.
  Same UV exposure as Condition 1.
  Modern cell UV response baseline.

Condition 4 — Modern yeast no UV:
  Germinating yeast culture.
  Standard YPD medium.
  No UV exposure.
  Modern cell baseline.
```

### What You Are Measuring

```
Primary measurements:

1. Cryptochrome gene expression:
   CRY1/CRY2 equivalent in yeast

   RT-qPCR or RNA-seq.
   Comparing Condition 1 vs 2
   (LECA UV vs no UV).
   Comparing Condition 1 vs 3
   (LECA UV vs modern UV).

2. Photolyase activity:
   Direct enzymatic assay.
   UV-damaged DNA substrate.
   Measure repair rate in
   cell extracts from each
   condition.
   LECA-grade vs modern cell.
   With UV vs without UV.

3. Cell viability:
   Does the LECA-grade cell
   survive UV exposure better
   than the modern cell?
   Or worse?
   Viability stain at each
   timepoint.
   Comparison across conditions.

4. Morphological response:
   Does UV exposure change
   the morphology of the
   LECA-grade cell?
   Does it trigger any response
   visible under microscopy?
   Phase contrast and DAPI
   at each timepoint.
```
