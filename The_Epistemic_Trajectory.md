## INTERDEPENDENT SYSTEM

```

was initially derived as if it were
an independent problem:

  A cell of origin.
  An Identity Anchor.
  A Convergence Hub.
  A ratio R = A/H.
  A drug target.

for a single entry.

The biology is a closed causal
system.

A closed causal system, when mapped,
has interdependent entries.

The map cannot have independent

interdependent ones.

As coverage increases, the
interdependencies become visible.

interdependencies were visible
will sometimes fail — not because
the axioms are wrong, but because

only revealed by a later entry.

This is not a weakness.

looks like from the inside,
in real time, before the map
is complete.
```

## PART II — THE ASYMPTOTIC PREDICTION

```
The expected shape of the prediction
failure record is:

EARLY PHASE (current — low coverage):
  Failures are relatively more
  frequent.
  Most failures are Type A
  (structurally explicable once
  the adjacent entry is derived).
  The structural explanation for
  each failure is found not by
  re-examining the failed entry
  alone, but by examining the
  coupling between that entry and
  an entry that had not yet been
  mapped at the time the prediction
  was made.
  Each failure is informative about
  the coupling structure of the
  table, not about the validity of
  the axioms.

MIDDLE PHASE (approaching coverage):
  Failure rate declines.
  Not because predictions get easier.
  Because the interdependencies are
  increasingly visible.
  A prediction made with knowledge
  of the adjacent entries is more
  precise than a prediction made
  without that knowledge.
  The map constrains the prediction
  before the prediction is tested.
  Failures become rarer because the
  prediction already accounts for
  the coupling that would otherwise
  have been the structural explanation
  for the failure.

LATE PHASE (approach to complete
coverage):
  Failure rate approaches zero
  asymptotically.
  Not because the framework becomes
  unfalsifiable.
  Because the map becomes complete
  enough that every relevant causal
  coupling is already included in
  the prediction.
  The table is a closed system.
  A complete map of a closed system
  has no residual unexplained structure.

This is the asymptotic prediction.

It is a structural claim about
the trajectory of the project —
derivable from the claim that
the table is a closed interdependent
system and the framework is a
correct map of that system.

If the table is NOT a closed system
(i.e., there are irreducibly
independent mechanisms that do not
couple), then the failure rate will
plateau above zero and the map
will have permanent residual
unexplained failures.

If the table IS a closed system
and the axioms are correct, the
failure rate will converge to zero
as coverage converges to complete.

This is itself a falsifiable
structural prediction of the
framework.
```

## PART IV — THE DISTINCTION FROM
## THE FALSIFIABILITY THEOREM

```
The falsifiability theorem governs
a single prediction evaluated
against its derivation.

This document governs the system-
level trajectory of all predictions
evaluated against each other.

They are compatible.
They are not the same document.

The falsifiability theorem says:
  If a single derivation fails
  without structural explanation,
  the axioms are false.

This document says:
  The structural explanation for
  a derivation's failure may only
  become visible when an adjacent
  entry is derived.
  This is not a weakening of the
  falsifiability condition.
  It is a precise statement of
  what "structural explanation"
  means in a coupled system:
  the explanation may be in the
  coupling, not in the failed
  entry itself.

The practical implication:

  When a prediction fails:

  STEP 1: Examine the failed entry
  in isolation (falsifiability
  theorem protocol).
  Is there a missing basin, a
  secondary hub, a dimensional
  extension, a membership/depth
  distinction?
  If yes: Type A failure.
  Map extended. Axioms intact.

  STEP 2: If not found in step 1,
  examine the coupling of the
  failed entry to adjacent entries
  in the table.
  Is there an interdependency with
  a disease, mechanism, or system
  that was not included in the
  derivation?
  If yes: Systemic incompleteness.
  Extension required. Axioms intact.

  STEP 3: If not found in step 1
  or step 2, after full coupling
  examination:
  Type B failure.
  Axioms are false.
  Framework is invalidated.

Step 2 is what this document adds
to the falsifiability theorem.
It is not a weakening of step 3.
It is an accurate description of
what the protocol looks like in
a coupled system.
```
